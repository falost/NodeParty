# nodeParty

 ## NodeJs 北京线下聚会 活动记录

### 第一期 （圆满结束）
    NodeParty线下活动由狼叔（i5ting）发起，波大（裕波）协办，共邀请5位讲师（justjavac，精子，@stonephp，张晋涛，胡戎），场地由360（裕波帮忙联系）赞助，茶歇由stuq赞助。
    活动于2017年07月29日（星期六）在360大厦举行，共计98人到场，活动进行时间为下午13:30点到晚上18:15刻，共计5个小时左右，各位讲师分享的内容精彩不断，掌声不断！
